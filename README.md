## Installation

```bash
git clone https://github.com/samiul-alim/photo-catalog.git
cd photo-catalog
yarn
```

## Get started

```bash
yarn start
```

## Build & Deploy
Configure firebase deployment before firebase deploy.

```bash
yarn build
firebase deploy
```

This web-app is built using [create-react-app](https://github.com/facebook/create-react-app)
